# Python Password Generator

A beginner-friendly Python project that demonstrates how to generate secure passwords using randomization, user input validation, and modular programming techniques.

This repository contains two versions of a password generator:

1. **Simple Password Generator** – Automatically generates a strong password with uppercase letters, lowercase letters, numbers, and symbols.
2. **Advanced Password Generator** – Allows users to customize password length and specify the number of letters, numbers, and special characters.

This project was created as part of my Python learning journey while exploring cybersecurity-related programming concepts.

---

## Features

### Simple Password Generator

* Generates strong random passwords
* Configurable password length
* Includes:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Special characters
* Automatically ensures password complexity requirements are met

### Advanced Password Generator

* Interactive menu-driven interface
* Fully random password generation
* Custom password generation
* User input validation
* Exception handling
* Character allocation verification
* Randomized character placement

## Installation

Clone the repository:

```bash
git clone https://github.com/ramiyakaru/python-password-generator.git
```

Navigate to the project directory:

```bash
cd password-generator
```

(Optional) Create a virtual environment:

### Windows

```cmd
python -m venv venv
venv\Scripts\activate
```

### macOS / Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```
